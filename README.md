# 《明日方舟：终末地》风格的个人主页
自己使用的个人主页网站源码，《明日方舟：终末地》风格，包含个人介绍、技术栈、游戏展示等模块

![效果展示](/img/preview_photo_1.png)

### 🎉 功能特点
- ✅  纯静态（HTML + CSS + JavaScript)
- ✅  响应式布局，适配移动端与桌面端
- ✅  滚动动画
- ✅  背景音乐
- ✅  可展开 / 收起侧边导航栏
- ✅  代码体积精简，依托多家公共 CDN，加载稳定

### ⚙️ 部署使用
项目属于纯静态前端页面，无运行环境依赖，可直接部署至任意静态站点服务：
- GitHub Pages、Vercel、Netlify
- Nginx、Apache 等 Web 服务器
> 字体、图标库等大体积文件已通过公共 CDN 网络分发，但项目中仍保留有图片及背景音乐等资源文件，为确保访客获得流畅的浏览体验，建议服务器出网带宽 **≥ 30 Mbps** ，或将站点接入 CDN 加速分发网络
